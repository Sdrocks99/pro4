What's the difference?

HTML

Hypertext markup language

Create the structure

    Controls layout of content
    Provides structure to webpage
    Fundamental building block to any web page

CSS

Cascading style sheets

Stylize the website

    Applies style to web page
    Targets various screen sizes
    Handles look and feel of webpage

Javascript

Increase interactivity

    Adds interactivity to web page
    Handles complex functions
    Programmatic code which enhance the functionality
